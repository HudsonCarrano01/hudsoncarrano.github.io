Site Description

The The Weather website is a weather information site.

Site Purpose

The main purpose of The Weather website is provide information about the weather and temperature allowing you to plan your day according to weather conditions. All information is hourly updated, giving you the conditions to decide whether to leave the house or not, if you take an umbrella, if you put on more cold clothes, if you go fishing, if you go sailing etc ...
